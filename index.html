INTRODUCTION

Deep Blue Business is a light multipurpose business WordPress theme that comes with a smooth design and extremely impressive functionality to unleash your website. It has been made to showcase any services that you provide. You can use it for any kind of website and deliver an awesome look. Deep Blue Business includes both the looks and astounding functionality to further intensify the website. The new multipurpose business theme, Deep Blue Business is packed with features like Featured Content, Hero Content, What We Do, Testimonials, and more. All of these features have been set to keep multipurpose business websites in mind. Deep Blue Business is the ultimate stop if you want a light and bold theme for your website.

HOW TO IMPORT DEMO

There is few steps to import the awesome fansee business pro theme. Please check out the below video.

Home page customizer

  • To customize the home page, first go to Dashboard then click on Appearance and click Customize.

    After clicking on "Customize", options like front page options, theme options, site identity, background image menus etc will be listed as shown in the figure below.

Front Page Option

  • Front page option helps to customize the look of front page. You can customize front page by customizing following options shown in figure.

General

  • This option helps to customize the general settings of the front page. It includes Show front page content.
  • Show Front Page Content: It helps to enable or disable the front page content.

    When we enable the "Show Front Page Content", another "Show Content Above" option gets enabled automatically. This option helps to enable or disable the content above the page.

Slider

  • This option helps to customize the slider settings of the front page.
  • Slider settings includes following features.
    • Enable Slider: It helps to show the slider on the front page.
    • Pages: It helps to add or delete the page shown in the slider of the front page as per requirement.
    • Autoplay: When enabled, it helps to play the slider of the page automatically one after another.
    • Dots: It shows number of dot shape below the slider based on the number of pages included in the slider.
    • Infinite Scroll: It helps to keep the slider on loop when enabled.
    • Enable Shortcode: It helps to generate shortcode when enabled.

Service

  • This option helps to customize the Service section settings of the front page.
  • Service section settings includes following features.
    • Enable: It helps to enable or disable service section on the front page. When service section is enabled, content page, sub text, sub pages, button text are also automatically enabled.
    • Content Page: It helps to choose the page for "Our Services" from the pages.
    • Sub Text: It helps to write sub text based on content page.
    • Sub Pages: It helps to create or delete sub pages on "Our Services" section as per requirement.
    • Button Text: It helps to write the text on the button available in "Our Services".
    • Enable Shortcode: It helps to generate shortcode when enabled.

About

  • This option helps to customize the about us settings of the front page.
  • After clicking on About, following features shows up.
    • Enable: It helps to enable or disable the "About us" section on the front page.
    • Page: It helps to select the page from the pages we have listed for About us section.
    • Sub Title: It helps to write the subtitle of the page in About us section.
    • Button Text: It helps to write text on the button available in About us section.
    • Enable Shortcode: It helps to generate shortcode when enabled.

Brand

  • This option helps to customize the brand settings of the front page.
  • After clicking on Brand, following features shows up.
    • Enable: It helps to enable or disable the "Brand" section on the front page.
    • Content Page: It helps to select the content page from the pages we have listed for Brand section.
    • Brand Pages: This option helps to add new brand page from the pages that have listed for Brand section and also helps to delete brand page as per requirement.
    • Enable Shortcode: It helps to generate shortcode when enabled.

Team

  • This option helps to customize the team section settings of the front page.
  • Team section settings includes following features.
    • Enable: It helps to enable or disable team section on the front page.
    • Content Page: It helps to choose the page for "Team" from the pages.
    • Sub Text: It helps to write sub text based on content page in "Team" section.
    • Button Text: It helps to write the text on the button available in "Team" section.
    • Sub Pages: It helps to create or delete sub pages for the team member on "Team" section as per requirement.
    • Team to show in frontpage: It helps to show number of team. Maximum number of team you can include is 4 for now.
    • Select a Team Archive Page: This option helps to select the team archive page when clicked on button available in "Team" section
    • Enable Shortcode: It helps to generate shortcode when enabled.

Achievement

  • This option helps to customize the achievement section settings of the front page.
  • Achievement section settings includes following features.
    • Enable: It helps to enable or disable achievement section on the front page.
    • Content Page: It helps to choose the page for "Achievement" from the pages listed.
    • Sub Text: It helps to write sub text in "Achievement" section above the content page.
    • Sub Pages: It helps to create or delete sub pages of the achievement as per requirement.
    • Enable Shortcode: It helps to generate shortcode when enabled.

Blog

  • This option helps to customize the Blog section settings of the front page.
  • Blog section settings includes following features.
    • Enable: It helps to enable or disable Blog section on the front page.
    • Content Page: It helps to choose the page for "Blog" from the pages.
    • Sub Text: It helps to write sub text in "Blog" section above the content page.
    • Total Column per Row: It helps to show how many column you can enter in a row for "Blog" section. Maximum number of blog you can include is 4 for now.
    • Enable Shortcode: It helps to generate shortcode when enabled.

Testimonial

  • This option helps to customize the testimonial section settings of the front page.
  • Testimonial section settings includes following features.
    • Enable: It helps to enable or disable testimonial section on the front page.
    • Content Page: It helps to choose the page for "Testimonial" from the pages.
    • Sub Text: It helps to write sub text in "Testimonial" section above the content page.
    • Sub Pages: It helps to create or delete sub pages for the testimonial on "Testimonial" section as per requirement.
    • Enable Shortcode: It helps to generate shortcode when enabled.

Call to Action

  • This option helps to customize the call to action section settings of the front page.
  • Call to Action section settings includes following features.
    • Enable: It helps to enable or disable "Call to Action" section on the front page.
    • Content Page: It helps to select the page that is to be shown in "Call to Action" section.
    • Contact Form Shortcode: It helps to create shortcode for contact form.
    • Enable Shortcode: It helps to generate shortcode when enabled.
